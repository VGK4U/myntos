Global ConfigurationAdmin Dashboard
VGK4U Member Modules (Phase 1 — Read-Only)
Each toggle controls whether the corresponding VGK4U-Members tab returns data on the matching staff admin page. Default: ON. Switching a toggle off hides VGK4U rows but keeps the existing MNR Members tab unaffected.
VGK4U Member Modules (Phase 2 — Write-Flow)
Each toggle controls a member-facing write surface (feedback, KYC, bank, settings, etc.). Default: OFF (Zero-Default-Access). When OFF, the matching /vgk/* page shows a "Module Disabled" overlay to members. Each change is recorded in the audit log.
