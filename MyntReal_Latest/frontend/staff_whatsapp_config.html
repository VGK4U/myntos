Meta Template Guidelines (live)
  • Name: lowercase letters, digits, underscores only — max 60 chars (e.g. vgk_welcome_v2)
  • Variables: must use {{1}}, {{2}} format — not {{name}}
  • Body: max 1,024 chars · Header text: max 60 chars · Footer: max 60 chars
  • MARKETING = promotional (1-5 min review) · UTILITY = transactional (instant) · AUTHENTICATION = OTP (instant)
  • No lotteries, medical claims, political content, or misleading offers
  • No raw URLs in body (e.g. https://...) — top cause of INVALID_FORMAT rejection. Use {{N}} variable or a CTA button instead
  • Rejection reasons appear in the Approval Status tab after syncing

How to get a Permanent Token

  1. Go to business.facebook.com
  2. Click Business Settings (gear icon)
  3. Under Users, click System Users
  4. Select existing System User or click Add to create one
  5. Click Generate New Token
  6. Select your App and check these permissions:
    • whatsapp_business_messaging
    • whatsapp_business_management
  7. Set Token Expiry to Never
  8. Click Generate Token and copy it here
Note: The token shown in Meta's API Setup page (Quick Send) expires in 24 hours. You must use a System User permanent token.
