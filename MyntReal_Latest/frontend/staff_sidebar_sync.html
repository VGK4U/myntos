Sync Sidebar Registry

Important: This action will update the database menu registry with the current 18-section sidebar structure. Use this after publishing to ensure production matches development.

This tool synchronizes the sidebar menu configuration from the codebase to the database. It ensures that:

  • All 18 sections are properly ordered (Progress through MNR User Sidebar)
  • Section names are updated (MNR, MNR USER SIDEBAR)
  • Menu items are correctly assigned to their sections
  • Legacy/duplicate sections are cleaned up

When to Use
  • After publishing new code to production
  • When sidebar sections appear incorrect
  • After menu structure changes in code
  • To fix legacy/duplicate sections
Access Control

Only VGK Mentor can perform this sync. The Zero-Default Access Policy is preserved - new sections won't automatically grant access to users.

Staff Impact

This sync only updates menu registry structure. It does NOT change any staff member's access permissions. Existing permissions in the Access Matrix are preserved.
